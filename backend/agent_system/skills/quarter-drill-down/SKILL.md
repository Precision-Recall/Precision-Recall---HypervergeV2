# quarter-drill-down

## Overview
Filters retrieval to quarterly 10-Q filings for granular analysis.

## Parameters
- `company` (str): Company name
- `year` (int): Filing year
- `quarter` (int): Quarter number (1-4)
- `query` (str): Topic to search

## Output
Top-10 reranked chunks from the quarterly filing.
